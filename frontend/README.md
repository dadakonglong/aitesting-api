# AI智能接口测试平台 - 前端界面

## ✅ 已完成

### 页面列表

1. **首页 - 场景生成** (`/`)
   - 自然语言输入
   - AI场景生成
   - 实时结果展示
   - 3个特性卡片展示

2. **测试管理** (`/tests`)
   - 场景列表展示

3. **数据导入** (`/import`)
